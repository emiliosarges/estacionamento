Desenvolvimento de uma API REST
para Sistema de Estacionamento

MVP sem autenticação, pagamentos ou integrações externas.
Foco em REST, DTOs, JPA, validações, regras de negócio, transações e tratamento de erros

1. Objetivo
Desenvolver uma API REST para controlar as operações essenciais de um estacionamento:
- cadastro de vagas;
- entrada de veículos;
- saída de veículos;
- consulta de veículos estacionados;
- cálculo do valor da permanência.
O sistema não terá autenticação neste primeiro momento.
